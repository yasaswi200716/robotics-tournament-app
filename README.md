#  Robotics Tournament Manager

A simple web application to manage robotics competitions like a sport.  
It allows organizers to register teams, update scores, and view a live leaderboard.

## Features
- Add and manage teams
- Update scores
- Automatically sorted leaderboard
- Persistent data using browser localStorage

## Tech Stack
- HTML
- CSS
- JavaScript
- localStorage

## Inspiration
Inspired by the idea that **robotics is a sport** and should have structured tournaments and scoreboards like traditional sports.

## How to Run
Open `index.html` in any modern browser and start managing a tournament.
